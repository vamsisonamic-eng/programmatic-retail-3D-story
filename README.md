# Retail Media, Decoded — A 3D Scroll Story

An immersive, scroll-driven interactive data visualization that breaks down the mechanics of Retail Media Networks (RMNs) and programmatic advertising. Built using high-performance 2D HTML5 Canvas acting as a 3D projection engine, this project avoids heavy third-party WebGL libraries to deliver 60 FPS fluid storytelling.

## ✨ Features
*   **Pure Canvas 3D Engine:** High-performance, allocation-free matrix rotations projecting a complex fractured cereal box in real-time without Three.js/WebGL.
*   **Synchronized Scroll-Driven Timeline:** Chapters fade, transform, and trigger specific physics transitions precisely indexed against the viewport scroll boundaries.
*   **Interactive Programmatic Simulator:** A live automated 60ms workflow simulator mapping out SSP, DSP, and verification networks step-by-step.
*   **Live Order Book Terminal:** Simulates a real-time programmatic auction clearing room, randomizing bidder fits and prices on every loop.

---

## 📈 Covered Insights
The presentation guides users through a multi-stage data journey:
1.  **The Box Metaphor:** How everyday product space transitions into active advertising mediums.
2.  **The Margin Flip:** Visualizing the massive delta between traditional grocery margins (~3%) and RMN ad margins (70–90%).
3.  **The Programmatic Engine:** Breaking down millisecond bidding auctions (DSP vs. SSP).
4.  **Global Case Studies:** Comparing market models across global quick-commerce ad booms.
5.  **The Privacy Vault:** Explaining data clean rooms and tracking cookie elimination.
